## 🏁 학급자율활동 동료평가 기준 {#peer}

### 1) 출결 점수 규정 (5점 만점)
- **미인정 지각/조퇴/결과 1회당 0.5점 감점**  
- **미인정 결석 1회 = 미인정 지각 3회로 처리(총 1.5점 감점)**

예시)
- 미인정 지각 2회 → 5점 − 1.0점 = **4.0점**
- 미인정 결석 1회 → 5점 − 1.5점 = **3.5점**

### 2) 1인 1역(기여)
- **학기말에 1회씩, 총 2회 실시**
- (주소 추후 안내) **구글 설문**으로 온라인 동료평가
- 평소 학급 친구들의 기여도를 관찰하여 평가

### 3) 주번 활동
- **매주 월요일(휴일인 경우 익일) 조회시간에 평가**

### 가산점 제도
- **동료평가의 평균 점수에 가산점을 더함**
- **가산점 = (주번 일수) × 0.1점**

예시)
- 동료평가 평균이 **4.2점**
- 한 주가 4일이라 주번 일수 4일이면  
  → **4.2 + 0.4 = 4.6점**

### 출결 주요내용 (자세한 내용은 가정통신문 참조)

<div class="callout warn">
  <div class="title">결석 처리 방법</div>
  <ul>
    <li><b>결석(질병/인정/미인정 모두 포함)</b>: 증빙서류(사진촬영)와 함께 학부모가 직접 <b>‘학부모 나이스 서비스’</b>로 결석 신고 → 담임이 처리</li>
    <li><b>결석이 아닌 지각/조퇴/결과</b>: <b>증빙 서류 원본</b> 제출</li>
  </ul>
</div>

#### 출결 증빙서류

| 사유 | 증빙서류(예시) |
|---|---|
| 질병 | 진료확인서 또는 (병원에서 발급한) 처방전 |
| 경조사 | 관련 증빙서류 |
| 대입 | 관련 증빙서류 |
| 감염병(코로나 등) | 격리통지서, 신속항원검사 양성확인서 등 병원 확진 확인서 |

- **증빙서류 제출 기한**: 결석 등 한 날로부터 **5일 이내** 제출(휴무일 제외)  
  - 예시) **월요일** 결석 등 → 늦어도 **(금)요일**까지 증빙서류 제출
- **증빙서류 인정 기간**: **해당일 포함 2일**(휴무일 포함)  
  - 예시) **(화)** 결석 등 → **(화)** 또는 **(월, 수)** 자 진료확인서/처방전 필요  
  - 예시) **(월, 수)** 결석 등 → **화요일 확인서로 이틀 결석 확인 인정? NO**

### ⚠️ 학급운영에서 주의할 것

- **학급 지각 08:00**을 지키자. *(학교 지각 08:10 기준과 다름)*  
  - 학급지각 시: 방과후 청소 또는 허드렛일  
- 서로 칭찬하는 학급 분위기 만들자 → **상점 + 세특 기록**
- **1인 1역(기여)**로 자율성과 공동체 미덕을 기르자 → **세특 기록**
- 방과후 개인 물건은 교실에 두지 않고 **의자는 올린다**  
  - 걸리면 방과후 청소 또는 허드렛일
- 자기 주변 청소는 자기가 하는 것이 원칙
- **조회와 종례는 매일** 한다(특별한 경우 제외)
- 등교 후 자기주도적으로 **“아침의 기적”** 실행 → 자기성장 + 학년말 세특기록
